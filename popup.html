A11Y Assist

From the accessibility perspective we need to implement following

  • At least one H1 tag: The document/webpage must contain at least one H1 tag, which describes the primary content or purpose of the page
  • Title tag: The document/webpage must contain a Title tag
  • Title & H1 relation: The title tag value must contain the h1 tag value. Expected H1 value to be in the title
